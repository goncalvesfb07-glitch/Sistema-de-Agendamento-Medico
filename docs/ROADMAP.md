# Fluxo de Desenvolvimento

Cada Sprint seguirá o seguinte processo:

1. Planejamento da Sprint
2. Criação da branch da Sprint
3. Desenvolvimento
4. Testes
5. Commit
6. Merge para a branch main
7. Atualização da documentação
